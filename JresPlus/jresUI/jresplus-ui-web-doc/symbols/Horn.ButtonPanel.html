按钮组件button

可用的宏组件有
button_panel:用于表单的提交按钮面板，布局上独占一行，内置提交和重置，可修改名称；
button_panel_ex:用于表单中的按钮面板，布局上独占一行，没有内置按钮；
button_group:用于表单中的按钮，布局上和不同的输入组件一致，可使用cols属性配置所占列数，没有内置按钮；

适用范围

用于表单数据的提交,和表单中的按钮

约束

  1. 无

注意事项

  1. button_group只能用于panel内部
  2. button_panel和button_panel_ex不能放在panel内部使用，使用方式和panel类似
  3. button_panel默认包含“提交”和“重置”按钮，用于表单的提交
  4. button_panel_ex不包含“提交”和“重置”按钮

属性

注意：组件的id、name属性只能是英文字母开始，由英文字母、数字和下划线组成，命名时应考虑易识别，不要太长，不要和其他组件重复（表单内的组件除外）。

名称 类型 描述 默认值

组件唯一标识
支持此属性的按钮容器组件(button_group,button_panel,button_panel_ex)

submitLabel String

表单中提交按钮的显示标签
如果要使用默认属性，不要设置此属性，更不要使用空字符串
支持此属性的按钮容器组件(button_panel)

resetLabel String

表单中重置按钮的显示标签
如果要使用默认属性，不要设置此属性，更不要使用空字符串
支持此属性的按钮容器组件(button_panel)

 "重置"
buttons Array[Json]

按钮组件中的自定义按钮配置
支持此属性的按钮容器组件(button_group,button_panel,button_panel_ex)
按钮的数据格式:
{"name":"btn1",//按钮组件名称
"label":"查询",//按钮显示名称
"className":"u-btn-white",//按钮自定义样式，非必须项。默认提供了u-btn-primary(蓝色)、u-btn-default(灰色)、u-btn-success(绿色)、u-btn-danger(红色)、u-btn-white(白色)和u-btn-warning(黄色)
"event":"lhkh.query()"//按钮被点击时触发的函数执行

"refmenu":"test"//该按钮关联的menu组件（菜单） }

参数:

  • boolean:enabled 如果为true设置为可用，设置为false，设置不可用,此参数不传入时默认为true;
    • 返回值:
  • void:
    • 示例：
