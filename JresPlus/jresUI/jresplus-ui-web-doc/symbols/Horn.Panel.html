Panel 面板组件

属于容器组件，做为单行多列组件的容器使用，多用于表单的容器

依赖组件

  • 无

适用范围

适用于表单内的容器组件

约束

注意事项

  1. panel组件本身的dom结构使用UL创建
  2. panel组件本身没有可视的内容需要做为容器与其他组件结合使用

属性

注意：组件的id、name属性只能是英文字母开始，由英文字母、数字和下划线组成，命名时应考虑易识别，不要太长，不要和其他组件重复（表单内的组件除外）。

名称 类型 描述 默认值
id String

组件的唯一标示
name String

组件的名称
title String

面板组件的标题栏,默认没有标题栏

 ""
cols int

列数（目前支持1-4列）

示例： 
#@panel({"id":"testspanel","title":"","cols":3})

#end
 3
expandable Boolean

是否启用展开/收缩功能,默认不启用

示例： 
#@panel({"expandable":true})
	#textfield({"label":"流程名称：", "name":"hsBpmNameLike"})
#end
 false
isExpand Boolean

是否展开(只有在expandable为true生效 ),默认不展开

示例： 
#@panel({"expandable":true,"isExpand":true})
	#textfield({"label":"流程名称：", "name":"hsBpmNameLike"})
#end
 false
