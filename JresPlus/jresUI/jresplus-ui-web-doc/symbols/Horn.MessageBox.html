MessageBox 消息提示

替代html的alert,confirm

  • 无

适用范围

仅提供js工具类，不提供宏组件

约束

注意事项

  1. 无

使用示例

alert

confirm

 

方法

名称 参数 描述
alert (title, text, fn)

只带确定按钮的提示窗，用于操作成功后的提示

参数:
  • String:title 显示标题
  • String:text 显示信息
  • Function:fn 回调函数
    • 示例： 
    Horn.Msg.alert("提示","点击了ok");或Horn.Msg.alert("提示","这个一个提示信息",function(){alert(1);});
    warning (title, text, fn)

    只带确定按钮的警告提示窗

    参数:
  • String:title 显示标题
  • String:text 显示信息
  • Function:fn 回调函数
    • 示例： 
    Horn.Msg.warning("提示","点击了ok");或Horn.Msg.warning("提示","这个一个提示信息",function(){alert(1);});
    error (title, text, fn)

    只带确定按钮的错误提示窗

    参数:
  • String:title 显示标题
  • String:text 显示信息
  • Function:fn 回调函数
    • 示例： 
    Horn.Msg.error("提示","点击了ok");或Horn.Msg.error("提示","这个一个提示信息",function(){alert(1);});
    confirm (title, text, okFn, cancelFn)

    带确定、取消按钮的确认对话框

    参数:
  • String:title 显示标题
  • String:text 显示信息
  • Function:okFn ok回调函数
  • Function:cancelFn 取消回调函数
    • 示例： 
    Horn.Msg.confirm("确认","是否确认",function(){
		Horn.Msg.alert("提示","点击了ok");
	},function(){
		Horn.Msg.alert("提示","点击cancel");
	})
