radio_group单选框组件

组件描述

继承组件

  • Horn.Field

适用范围

当选择项不是太多，不会造成布局混乱时使用。如果选择项太多时，可以考虑增大cols值的设置(有效值:1,2,3) 或使用combox组件替换

约束

  1. 无

注意事项

  1. name属性必须设置，否则会造成组件无法正常使用！
  2. 数据项不易过多（一行内可以完整显示），否则会导致换行显示影响美观，如果需要显示更多的项，可以考虑使用combox组件
  3. 需要使用panel组件做为容器

方法

名称 参数 描述
addRule (rule)

增加校验规则

参数:
  • String:rule 校验规则字符串
    • removeRule (rule)

    删除校验规则

    参数:
  • String:rule 校验规则字符串
    • show ()

    显示表单，如果表单已经显示，此方法无效果，hide方法与之相对应
    hide ()

    隐藏表单，如果表单已经隐藏，此方法无效果，show方法与之对应
    setRequired (required)

    设置为必填项，同时增加红色的 *

    参数:
  • Boolean:required 不传值或者传true表示必选项，传false表示取消必选项
    • setDisabled (disabled)

    设置字段是否禁用，被设置为禁用的组件，不可以编辑，也不参与表单提交，但是可以通过API的方式修改表单的值

    参数:
  • Boolean:disabled true表示禁用，false表示正常
    • setValue (value)

    设置表单的值

    参数:
  • String:value 值
    • getValue ()

    获取表单的值

    返回值:
  • : 表单的提交值
    • reset ()

    如果设置了defValue的值，重置成的defValue值，否则重置成value值
    clearValue ()

    清空表单的值，显示值和隐藏值都设置为""
    isValid ()

    获取由validate方法触发表单校验后的结果，并通过返回值标识校验的结果

    返回值:
  • Boolean: true表示校验通过，false表示校验失败
    • validate ()

    触发校验表单的内容，然后通过调用isValid方法获取校验的结果
