弹窗组件 window

用于弹出窗口展示内容，属于容器组件,包含标题以及标题栏部分的关闭按钮，没有具体的内容。

依赖组件

  • 无

适用范围

使用于内容较少，且隶属与一个大的页面，在交互上与大页面存在关联，比如“新增一条数据记录以弹出窗口的形式展示，保存后刷新主页面”

约束

  1. 使用window标签时，必须放在screen标签内部
  2. 不适合展示内容较多的页面，因为窗口太大不符合审美，且可能出现不友好的窗口内滚动条

注意事项

  1. 窗口中的内容最好在主页面初始时就存在，不建议使用异步方式加载渲染
  2. 慎用窗口内的iframe,会导致复杂
  3. 慎用窗口的高度和宽度需要根据实际的展示内容，进行设置，设置时注意最大值和最小值

事件

名称 参数 描述
beforeClose function()

窗口关闭前回调事件
如果回调函数返回false,则窗口终止关闭动作

示例： 
#@window({"name":"w6","title":"test","beforeClose":"beforeC()"})
    	
    #end
    function beforeC(){
    	alert("关闭之前");
    	return true;
    }
afterClose function()

窗口关闭后回调事件

示例： 
 #@window({"name":"w6","title":"test","afterClose":"afterC()"})
    	
    #end
    function afterC(){
    	alert("关闭之后");
    }
