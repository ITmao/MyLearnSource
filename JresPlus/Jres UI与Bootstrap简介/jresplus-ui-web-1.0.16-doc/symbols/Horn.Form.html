方法

名称 参数 描述
submit ()

实现form提交
reset ()

重置form内部所有非禁用状态表单的值
clearValue ()

清空form内部所有表单的值
getValues ()

获取form的值，以名值对的方式返回

返回值:
  • : Object
    • 示例： 
    如
var form = Horn.getComp("form") ;
form.getValues() ;
结果--
{"client_id":"1000012","fund_account":"2000112"}
    getFieldComps ()

    获得Form中的field，返回的是一组Horn.Field对象。

    返回值:
  • : Horn.Field对象数组
    • setValues (|{Array}, prefix, clear)

    给form表单元素设置值

    参数:
  • Object:|{Array} value 设置的值，
  • String:prefix 名称前缀，用于设置'client.name'这一类的值，那么前缀必须跟上"."，即为完整值为"client."才能正常使用， 对于没有前缀的表单，prefix会被忽略。
  • Boolean:clear 若未找到是否设置为空，如果只不为false则表示清空
    • 示例： 
    如
var form = Horn.getComp("form") ;
form.setValues({"client_id":"1000122","fund_account":"112222221"}) ;
