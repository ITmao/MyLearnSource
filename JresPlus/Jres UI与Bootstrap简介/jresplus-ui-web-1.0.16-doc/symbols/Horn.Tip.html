Tip 信息提示

从界面的上方显示的消息提示，提示后较短时间后自动隐藏，如果有条信息，新的信息会在最上方展示

适用范围

适用于，不需要进行操作和处理的消息提示，提示后在较短的时间里会自动隐藏或通过点击提示框马上隐藏,用来提示的信息应该是一句简短的文字，不宜过长（建议在20个字符以内）

注意事项

  1. 无

方法

名称 参数 描述
info (message, ms)

显示提示消息

参数:
  • string:message 提示消息信息
  • Number:ms 提示框自动隐藏的时间以毫秒为单位，若ms不为空，隐藏时间为：2000毫秒(提示框停留时间)+ms(提示框渐变消失时间),否则默认为2000毫秒(提示框停留时间)+500毫秒(提示框渐变消失时间)。
    • 返回值:
  • void:
