combox

替代html的select组件，有更加丰富的交互和功能

继承组件

  • Horn.Field

适用范围

替代原生html的实现，支持单选和多选，也可以绑定数据词典，但是不支持筛选

约束

  1. 无

注意事项

  1. 如果需要自定义code和text，codeField属性和textField必须同时设置
  2. 需要使用panel组件做为容器

    删除校验规则

    参数:
  • String:rule 校验规则字符串
    • show ()

    显示表单，如果表单已经显示，此方法无效果，hide方法与之相对应
    hide ()

    隐藏表单，如果表单已经隐藏，此方法无效果，show方法与之对应
    setRequired (required)

    设置为必填项，同时增加红色的 *

    参数:
  • Boolean:required 不传值或者传true表示必选项，传false表示取消必选项
    • setDisabled (disabled)

    设置字段是否禁用，被设置为禁用的组件，不可以编辑，也不参与表单提交并且其所有校验状态都会消失，但是可以通过API的方式修改表单的值

    参数:
  • Boolean:disabled true表示禁用，false表示正常
    • setReadonly (readonly)

    设置是否只读，设置为只读方式的组件，不可以编辑，可以通过下拉选择改变值，可以通过setValue、reset等API修改表单的值，并且可以参与表单提交(注！readonly属性仅对多选的情况下有效，单选无效)

    参数:
  • Boolean:readonly true表示只读，false表示正常
    • setValue (value, triggerChange)

    设置表单的值

    参数:
  • String:value 值
  • Boolean:triggerChange 是否触发值更改事件
    • getValue ()

    获取表单的值

    返回值:
  • : 表单的提交值
    • reset ()

    如果设置了defValue的值，重置成的defValue值，否则重置成value值
    clearValue ()

    清空表单的值，显示值和隐藏值都设置为""
    isValid ()

    获取由validate方法触发表单校验后的结果，并通过返回值标识校验的结果

    返回值:
  • Boolean: true表示校验通过，false表示校验失败
    • validate ()

    触发校验表单的内容，然后通过调用isValid方法获取校验的结果
